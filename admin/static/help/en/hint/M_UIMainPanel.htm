MainPanel

The MainPanel tab shows 6 KPIs, a list of top busy sessions and a list of the latest database log for giving you a quick overview of real-time performance status of the monitored database server. This tab is always shown in the Content Area and refreshed automatically for providing real-time performance status.

From these charts and information, you may need to drill down or get other related information from items on the SideBar to find out the root cause if any unusual situation has been observed.

For instance, if the VP Busy% is keeping high and close to the CPU Busy %, you may need to take any of the following procedures.

1. Right-click VP Busy% and select any option from the context menu to get further virtual process-related information
2. Select an item related to virtual process or session like Top Busy Session or Top Running SQL from Real-Time Performance Analysis on the SideBar to get more information.
