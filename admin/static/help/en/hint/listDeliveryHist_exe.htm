Alert Delivery History

It shows alerts/notifications sent by the dbSonar Control Center for events detected on the selected dbSonar Agent*.

As a system-defined or user-defined event is detected, an alert or notification will be sent by email or SMS to notify persons who should manage the detected event. Any unsuccessfully delivered alert/notification will be re-sent again and again till it's delivered successfully.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.

You may drill down into Get Event Notification Recipient List to get the list of recipients who receive the delivered alert notice, as well as contact information of every recipient in the list.

Here are descriptions of columns in the list.

Column Description
ID Identification of event
Delivered Time Time when the alert is delivered successfully
dbSonar Agent Name of the dbSonar Agent where the event is detected
IP IP address where the database instance resides on
Instance Name of the database instance that the event occurred
Event Brief description of the event
Severity Level of urgency for the event
Type Type of the delivered alert
Method Indicates how the alert is delivered
Recipient Name of the recipient who recieves the alert notice
Target The destination where the alert is delivered to, like email address
Retry The number of times that the alert has been tried before delivered successfully
Status Indicates the delivery status
