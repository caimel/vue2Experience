Network Connections

Depending on your drilling-down selection, All Network Connections or Network Connections in Waiting, this page will display the connection and transmission information of each network connection on the host* or the information of network connections in waiting status only respectively. The information helps you identify possible abnormal connection or abnormal network situation.

The information consists of the following two sections:

 

The first section shows the status of each network connection, how many ports are used, what port number and IPs are used for each connection and how many bytes have been received/sent.

The second section shows the statistics of OK/error/dropped/overrun packages sent/received through each network interface.
