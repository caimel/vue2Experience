Glossary

 
 
Terminology Description
Cooked file A file sits on the filesystem of your operating system
dbSonar Agent A descriptive name of the DBSonar instance that monitors the target Informix server
Host The monitored server
Monitored database server The monitored informix instance on the monitored server
Smart Large Object A special datatype for holding a very large amount of data that is stored in the dbspace, sbspace.
SQL activity It presents a SQL skeleton extracted from similar SQL statements.
SQL execution tree The SQL execution plan represents in a tree structure.
SQL operator node A node on the SQL execution tree. Each node performs a specific action that may be iterated more than once in the node.
Thread A thread is a task for a virtual processor to service requests from client applications.
