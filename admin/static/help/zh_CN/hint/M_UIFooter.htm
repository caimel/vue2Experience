Footer

It is on the bottom of the dbSonar interface. Constant tools are provided on the footer to refresh and print the content of the current tab. If a tool is not supported on a tab, it will be gray-out or disappear.

Here is the description of each constant tool.

A timer tells you how many seconds left before the content of the current tab is refreshed. It shows on as you click .

It is a refresh button. It refreshes the content of the current tab upon your request. Each time you click it, the content of the current tab will be refreshed.

It is a refresh-toggle button for enabling/disabling auto-refreshing the content of the current tab. As the button is on, its timer is on and the refresh button will be off. The information will be refreshed when the timer counts down to zero. The timer will continue counting in the background even if the button is off.

It is to print the content on the current tab.

A tab like MainPanel is designed to refresh automatically all the time, when the tab is launched, its timer is on and shown, and both refresh and refresh-toggle button are disabled.

In a tab like Process List, if the number of returned rows exceeds the number of rows per page to display, a page navigator as below will show on the footer. The page navigator tells you which page you are viewing, the total number of rows, and what the begin and end row are for the current page.

Here is the description of each button on the page navigator:

Navigates you to the 1st page.
Navigates you to the previous page.
Navigates you to the next page.
Navigates you to the last page.
