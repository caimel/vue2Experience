Chunk I/O Information

This page shows size and disk I/O status of each chunk assigned to the monitored database server*. Information exhibited on this page includes dbspace name, dbspace type, allocated and used size of data storage, the size and times of disk I/O occurred on a chunk and the physical location of a chunk. According to chunk I/O information, you know better about whether it's necessary to improve disk I/O of a cooked file* via direct I/O or metadata I/O of smart large objects* for a disk I/O problem.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.
