Database Information

This page shows information of all databases created on the monitored database server*.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.

If you would like to perform UPDATE STATISTICS on a database, you can get the suggested script of a selected database via Generate Update Statistics Script option in the right-click menu.


Column Description
Database Name of the database
Owner Creator of the database
Created Date that the database was created
Logging Indicate whether logging is active
Buffer_Log Indicate whether buffer logging is active
Is Ansi Indicate whether the database is ANSI-compliant
NLS Supported Indicate whether the database is GLS-enabled
