Event Notification List

This page shows a list of notifications of the selected event captured by all the dbSonar Agents* that you have authority to access through dbSonar.

As a system-defined or user-defined event is detected, an alert or notification will be sent by email or SMS to notify persons who should manage the detected event.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.

Column Description
Event Name of the event that was detected
dbSonar Instance Name of the database instance that the event occurred
IP IP of the database server that the event occurred
Serverity Level of urgency for the event
Type Type of the delivered alert
Recipient Name of the recipient who recieved the alert notice
Target The destination where the alert was delivered to, like email address
Time The time when the alert was delivered successfully
Status The delivery status
