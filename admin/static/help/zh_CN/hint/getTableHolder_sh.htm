Table In Used

This page shows sessions that are holding tables/indexes or a specific table in all databases or in a specific database on the monitored database server*. Each session is displayed with its session ID, connection user, database of opened table/index, opened table/index etc.

If you notice a table that is held by a lot of sessions and causes the performance issue, you may need to diagnose and analyze the placement, extents and schema design of the table to improve performance.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.
 

   Criteria Window

 

According to the database and/or the table you specify in the criteria window below, the page will display sessions that hold tables/the specified table in the specified database or all databases. If no database and table is specified, the page will display all tables/indexes being held in sessions on all databases.
