List Analysis Jobs

All sessions that are analyzing, have been analyzed or have been stopped analyzing on the database server are listed in descending order of the column Analysis Start Time.

Each analysis process is displayed with when the analysis process started, how long the analysis process has been running/run, what the analysis status is and which session is analyzed in the process etc. Analysing is shown in the status when an analsis process is running, AP Completed is shown when an analysis process has done, and Analysis terminated is shown when an analysis process is terminated manually by the Stop Analysis Session option in the right-click menu.

You may drill down from a session with "AP Completed!" or "Analysis terminated!" status to List Analysis Result to view the analysis result. The analysis process of a session continues till the session is finished. You may manually stop the analysis process of a long session to get partial analysis information of the session by right-clicking the process and selecting the Stop Analysis Session option from the right-click menu.

An analysis process along with its analysis result can be removed from the list as right-clicking the process and selecting the Clear Analysis Result option from the right-click menu; while all analysis processes on the monitored database server* can be cleared up as selecting the Clear All Analysis Results option from the right-click menu.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.
