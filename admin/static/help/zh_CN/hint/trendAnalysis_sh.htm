Trend Analysis

Any item with prefix "Trend - " in the SideBar will launch a trend analysis tab. And each trend analysis tab is launched with a specific trend chart which is specified on its menu context. For example, the specific trend chart used in Trend - CPU Busy Rate(Max) is CPU Busy Rate(Max).

The trend chart of a specific item drawn between the Start and the End is helpful for identifying and tracing possible performance issues that are happening or had happened on the monitored server.
 

Symptom and Approach

If there is any unusual situation shown on the trend chart, you may right-click the point on the trend chart and drill down into SQL Point in Time to check what SQLs are running around the point in time and how these SQLs are running.
 

How to Operate

   View More Trend Charts

 

All trend charts provided by dbSonar are listed and classified into various groups. You may select or unselect an item in the Trend List by clicking the checkbox in front of each item. The trend chart of the selected item will be displayed automatically according to the same period of time and time mode. When a trend analysis tab shows initially, the corresponding item of its specific trend chart is checked and its group is expanded.

For quick removing all selected items, you can click Clean all chart button to uncheck all selected items and remove all displayed trend charts.

   Redrawing With New Time Period / Mode

 

You can apply different period of time and/or sampling rate to all displayed trend charts for macro-/ micro-viewing them. After specifying the expected values in Time Mode, Start and End, click Redraw button to redraw all displayed trend charts.

Time Mode, the sampling rate, indicates that trend charts are drawn with samples taken under this rate. There are 3 options: 5 Minutes, Hourly and Daily.

Start and End are used to specify the period of time to draw trend charts. You may click to pick the start and the end dates separately or enter the start and the end dates manually. Then click Redraw to redraw trend charts.

   How to Operate Trend Chart

 

When you click any place on trend charts, the red line shows as below.

The point-in-time values under the red line on diagrams are displayed in the Legend-Value table as below. They are listed in diagram order from top to bottom. The date and time of the red line shows above the Legend-Value table. For a diagram with more than one trend line representing separate cpu, device, DBSpace etc, the legend-value pair of each trend line will be displayed separately in the Legend-Value table with the same prefix.

For example:.

  • "CPU Busy Rate(Avg)" is the legend for the line on the "CPU Busy Rate(Avg)" diagram.
  • "CPU I/O Wait" is a legend prefix for each line on the "CPU I/O Wait" diagram. A line represents a CPU.
