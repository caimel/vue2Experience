Virtual Process List

Monitoring virtual processors listed on this page to determine if the number of virtual processors configured for the database server is optimal for the current level of activity. The definition of each column is described below.

Column Description
VP ID Identification number of a virtual processor
VP Class It's the class of a virtual processor. All VPs in a particular VP class are dedicated to processing the same type of threads*.
User Time(Sec) The total user time, in seconds, that this class of VP has spent running on the CPU.
System Time(Sec) The total system time, in seconds, that this class of VP has spent running on the CPU.
Ready Queue The number of threads in the ready queue of the VP that is ready to run
