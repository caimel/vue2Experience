SideBar

It's on the left portion of the dbSonar interface and contains 6 major categories : DB System Optimization, Real-Time Performance Analysis, Historical Performance Analysis, dbSonar Report, Event Manager and About. For Real-Time Performance and Historial Performance Analysis, you may diagnose and analize problem from system, session and/or SQL point of view. According to what you would like to watch, diagnose and analysis, you can locate a proprer function easily from these categories and groups under each category.

As clicking on an item, a new tab for the item is launched into the Content Area.

  • DB System Optimization - This category provides various database server statuses for you to diagnose and determine if any adjustment is required to improve the performance of the database.

  • Real-Time Performance Analysis - This category provides real-time information of current system processes, database sessions and SQL activities on the monitored database server* for diagnosing a performance issue that is happening.

  • Historical Performance Analysis - This category provides statistics and trends of system resources and database activities in a long period of time for analyzing and find out the root causes of performance issues that have occurred.

  • dbSonar Report - It provides complete health-examination reports of the entire database server for preventive management and capacity planning.

  • Event Manager - It allows you to view processing statuses of events captured by dbSonar Agents and delivery statuses of alerts sent by dbSonar.

  • About - It gives you a general information of the dbSonar system, such as license information.
