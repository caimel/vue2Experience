OS Disk Panel

This page exhibits disk busy rate, read/write size and average I/O waiting time in the period between the Start and the End, and lists current file system utilization sorted by the column Use%. The information will be helpful for identifying and tracing disk I/O related performance issues that are happening or had happened on the monitored server.
 

Symptom and Approach

If the disk is keeping a high utilization rate (exceeding 80%+) and average I/O waiting time is high, and the DB Disk RW / Sec on the MainPanel is high, you may need to find out what SQL activities are executing and how the resource has been taken.

a. Right-click the high utilization rate on the diagram and drill down into SQL Point in Time to check what SQLs are running and how SQLs are running in the interval of the red line.
b. Use the Trend Analysis to view other trend charts for getting more information of database read/write activities and dbspace read/write activities etc.
c. Use the Top I/O Session to check which session has the most I/O activities and what SQLs are executing by the session.
d. By monitoring the information in the File System Disk Space Utilization table, you may plan your database capacity and can take proactive action to add disk drives before the system runs out of disk space, since insufficient disk space can have a catastrophic impact on the database system.
 

How to Operate

   Trend Chart

 

When you click any place on the diagram, the red line shows as below.

The point-in-time values under the red line on diagrams are displayed in the Legend-Value table as below. They are listed in diagram order from top to bottom. The date and time of the red line shows above the Legend-Value table.

Here we briefly annotate legends appeared in the Legend-Value table.

  • "Disk I/O Busy Rate" is the legend for the line on the "Disk I/O Busy Rate" diagram.
  • "Disk Reads " and "Disk Writes" are legends for lines representing the read and write size respecttively on the "Disk Read/Write(KB/sec) " diagram.
  • "Disk I/O Wait " is the legent for line on the "Disk I/O Wait " diagram.

If you would like to view trend charts over a long period of time, please click to pick the start and end dates separately or enter the start and end dates manually. Then click Refresh to redraw diagrams.
