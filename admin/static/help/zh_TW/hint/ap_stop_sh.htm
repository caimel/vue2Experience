Stop Analysis Session

All sessions that are analyzing, have been analyzed or have been stopped analyzing on the monitored database server* are listed in descending order of the column Analysis Start Time. Each analysis process in the tabular list displays the information of the process iteself, which session is analyzed by the process, and resource utilization of the session etc.

As you select Stop Analysis Session option in the right-click menu of an analysis process, the analysis process will be stopped and its status will become "Analysis terminated!".

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.

The status of an analysis process will show "AP Completed! " only when the analysis process is completed, that is, the analyzed session is over. As you manually stop an analysis process via Stop Analysis Session option in the right-click menu of the analysis process, its status will become "Analysis terminated".

You can drill down a session with "AP Completed!" or "Analysis terminated!" status to List Analysis Result to view the analysis result.

An analysis process along with its analysis result can be removed out of the list by right-clicking Clear Analysis Result; while all analysis processes on the monitored database server* can be cleared up by right-clicking Clear All Analysis Result.
