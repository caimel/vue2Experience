Chunk Information

Monitoring usage of all chunks assigned to the monitored database server* in order to determine if extra chunk is required for a DBSpace. For each chunk, you can easily tell the total size, free size, usage percentage in bar chart and its physical location. Besides, you will have an idea of the total number of chunks in the database server and the number of chunks for each DBSpace.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.
