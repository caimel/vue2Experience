DB Parameter Analysis Report

All the RDBMS configuration parameters defined in the Informix server are listed along with their effective values and default values. Since performance tuning is an iterative process, as time goes by, you need to re-analyze these values and reconfigure them on a regular basis to maintain the best performance.

Here are descriptions of columns in the report

Column Description
DB Parameter Name Configuration parameter name
Effective Value Value currently in use
Original Value Value in the ONCONFIG file at boot time
Suggestions Provides helpful hints and clues for changing the original value
