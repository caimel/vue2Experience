SQL Evaluator

This page will present the execution plan and cost of the SQL statement on the specified database for allowing you to analyze whether any index is required to be created or modified or whether the adjustment of the SQL statement is required for lowering the execution cost. Besides, you may check the statistics of tables referred in the SQL via Outdated Update Statistics Table to verify if any update statistics is required.
 

How to Operate

 

The first portion on the page as shown below is where you can modify or enter a SQL statement for evaluation.

If you would like to evaluate the execution cost of the adjusted SQL statement before implementing the change into the application program, you can modify the SQL statement here, then click Evaluate SQL button to get the execution plan and cost of the adjusted SQL statement.

If you would like to evaluate the execution cost of a new SQL statement before implementing it into an application program, you can write the SQL statement here and select a database for evaluating the statement, then click Evaluate SQL button to get the execution plan and cost of the new SQL statement.
