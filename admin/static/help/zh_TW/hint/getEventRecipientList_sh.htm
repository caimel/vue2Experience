Event Notification Recipient List

Recipients that are supposed to receive the notification of the system-defined or user-defined event detected on the monitored database server* are shown along with their emails and/or cell phone numbers.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.


Column Description
Event Name of the event that was detected
Login Name Login name of the recipient
Last Name Last name of the recipient
First Name First name of the recipient
E-Mail The email address of the recipient
Cellphone The cell phone of the recipient to get the SMS of the event
