Event List

Depending on your selection on the SideBar, Unresolved Event List/ Resolved Event List / Event List or All Unresolved Event List / All Resolved Event List / All Event List, you will see a list of unresolved / resolved / both events captured and recorded by the watched dbSonar Agent* or all dbSonar Agents that you have authority to watch in dbSonar.

As a system-defined or user-defined event is activated and has not been taken care of, it will be displayed in the unresolved list. An event will be moved into the resolved list when its situation has been resolved by the administrator or is not detected any more.

To view more detailed causes of an event, you can right-click the event and select the Show Detected Event Details option from the right-click menu. To view more details about how an event was resolved, you can right-click the event and select the Show Resolved Event Details option from the right-click menu. To get notification information of an event, you can right-click the event and select the Get Event Notification List option from the right-click menu.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.


Column Description
ID Identification of event
Occurrence Time when the event occurred
Resolution Indicates that the event hasn't been resolved, or when it was resolved
dbSonar Agent Name of the dbSonar Agent where the event was captured
Severity Urgency level of the event
Event Brief description of the event
Status Indicates the status of the event such as opened, resolved etc.
Remark Comment that is entered when marking the event resolved manually
 

   Marking an event resolved

 

Please follows steps below to mark an event resolved and enter your own remark.

1. Launch Unresolved Event List or All Unresolved Event List from the Event Manager on the SideBar.
1. Check the box in front of the row that you want to mark as resolved.
2. Click Marked Event as Resolved.
3. Enter any comment if you want, then click Submit; otherwise just click Submit.
4. Click OK for the confirmation; otherwise this event won't be marked as resolved and stays in the unresolved list.

If you select more than one row to be marked as resolved, the remark if any will be applied to all of them.
