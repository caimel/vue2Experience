Object-Level Trend

A drilling-down option with prefix "Object-Level Trend - " will lauch a new tab to display the specific trend chart for a specified object. For example, the Object-Level Trend - DBSpace Page Reades/Writes will display the page reads/writes trend chart for a specified DBSpace that you select in a list.

The trend chart is helpful for identifying and tracing possible performance issues that are happening or had happened on the monitored server during the period between the Start and the End. The default period of a trend chart is from now up to 3 days ago.
 

Symptom and Approach

If there is any unusual situation shown on the trend chart, you may right-click on the trend chart and drill down into SQL Point in Time to check what SQLs are running and how SQLs are running in the period.
 

How to Operate

   View More Trend Charts

 

When a object-level trend analysis tab shows initially, the specific trend chart is checked in the Trend List and the corresponding group is expanded.

All trend charts provided by dbSonar are listed and classified into various groups in the Trend List. You may select or unselect an item in the Trend List by clicking the checkbox in front of each item. As the item is selected, the corresponding trend chart is displayed automatically based on the same period of time and time mode.

For quick removing all selected items, you can click Clean all chart button that will uncheck all selected items and remove all displayed trend charts.

   Redrawing With New Time Period / Mode

 

You can apply different period of time and/or sampling rate to all displayed trend charts for macro-/ micro-viewing them. Click Redraw button after specifying the expected values in Time Mode, Start and End to redraw all displayed trend charts.

Time Mode, the sampling rate, indicates that trend charts are drawn with samples taken under this sampling rate. There are 3 options, 5 Minutes, Hourly and Daily.

Start and End are used to specify the period of time to draw trend charts. You may click to pick the start and end dates separately or enter the start and end dates manually.

   How to Operate Trend Chart

 

When you click any place on trend charts, the red line shows as below.

The point-in-time values under the red line on diagrams are displayed in the Legend-Value table as below. They are listed in diagram order from top to bottom. The date and time of the red line shows above the Legend-Value table. For a diagram with more than one trend line representing separate cpu, device, DBSpace etc, the legend-value pair of each trend line will be displayed separately in the Legend-Value table with the same prefix.
