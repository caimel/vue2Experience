Locked Waiting Connections

This page displays all threads currently waiting on a condition. You are abe to know if the condition has been in place for too long and needs your intervention or if it can be ignored until it is removed automatically.

You can click a column head to sort the list in ascending/descending order of the selected column and click or anytime to refresh information on the page.


Column Description
Condition ID Identification of condition
Address In-memory address of the condition control block
Name Name of the condition
Waiter ID of the thread that is waiting on the condition
Wait Time The amount of wait time, in seconds, that the thread has been waiting on this condition
