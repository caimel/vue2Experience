Event Details

If you drill down into the Show Detected Event Details option from a unresolved/resolved event, you will see the brief information of the database server monitored where the event happened, the log information in the past couple of minutes before the event occurred, and when the event occurred.

If you drill down into the Show Resolved Event Details option from a resolved event, you will see the information of the resolved event and when the event was resolved.
