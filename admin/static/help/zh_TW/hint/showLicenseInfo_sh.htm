License Info

This page shows information of all valid licenses on database servers that you have authority to view. Each license information specifies which dbSonar instance, which host, which host IP and which database server the license is for, and how many days left before the license expires.

Here are descriptions of columns in the list.

Column Description
Config Name Name of the dbSonar instance*
Server Machine Name of the host machine
Host IP IP of the host machine
DBDatabase Server Name of the monitored database server*
Days before expiration Days left before the license expires

You can click a column head to sort the list in ascending/descending order of the selected column.
